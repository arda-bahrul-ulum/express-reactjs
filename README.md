### Setup back-end

```console
 npm install express
 npm install g nodemon
 npm install cors
 npm install bodyparser
 npm install prisma
 npx prisma init
 npm install @prisma/client
 npm install jsonwebtoken
 npm install expressvalidator
 npm install bcryptjs
```

```console
npm create vite
npm install axios
npm install jscookie
npm install reactrouterdom
```
